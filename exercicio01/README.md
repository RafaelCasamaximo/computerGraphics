Como utilizar o programa:

Build:
cd atividade01
sudo cmake .
sudo make

Usage:
./main PathDaImagem ModoDeConversao

ModosDeConversao:
0 - BGR -> CMY
1 - BGR -> CMYK
2 - BGR -> HSI

Nota: para melhor visualizacao do HSI os valores de HUE foram interpolados de 0-360 para 0-255
